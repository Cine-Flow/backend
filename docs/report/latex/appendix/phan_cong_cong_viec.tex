Bảng dưới đây liệt kê chi tiết phân công nhiệm vụ của 3 thành viên nhóm theo từng giai đoạn của đồ án CineFlow, đảm bảo sự cân bằng về khối lượng công việc khi mỗi thành viên đều đảm nhiệm song song cả phát triển giao diện (Frontend) lẫn logic nghiệp vụ (Backend) cho các cấu phần tương ứng:

  \hline
  \endhead

  \hline
  \endfoot

  1 & \textbf{Trần Sĩ Cường} \newline
  MSSV: N22DCCN013 \newline
  (Trưởng nhóm) &
  \textbf{1. Nhiệm vụ chính:} \newline
  \textbullet\ \textbf{Premier League (Bóng đá):} Thiết kế giao diện lịch thi đấu (\texttt{PremierLeagueFragment}); viết API/Service kết nối cổng dữ liệu API ngoài (\texttt{FootballDataClient} kèm cache), ánh xạ highlight bóng đá dựa trên trường \texttt{badge}. \newline
  \textbullet\ \textbf{More \& Account (Cá nhân):} Xây dựng giao diện trang cá nhân (\texttt{MoreFragment} / \texttt{AccountActivity}), các Custom Dialog bo góc điều chỉnh hồ sơ thông tin và đổi mật khẩu; lập trình API cập nhật Họ tên, Số điện thoại và đổi mật khẩu. \newline
  \textbf{2. Nhiệm vụ phối hợp:} \newline
  \textbullet\ \textbf{MinIO Storage:} Cấu hình dịch vụ Docker MinIO, viết API upload media (\texttt{/api/v1/files/upload}) và script PowerShell seed media. \newline
  \textbullet\ \textbf{Bảo mật & Thống kê:} Triển khai Token Rotation \& Theft Detection ở Backend và \texttt{TokenAuthenticator} ở Client. Lập trình giao diện biểu đồ Thống kê cá nhân (\texttt{UserAnalyticsActivity}) và API tổng hợp dữ liệu thời lượng/thể loại xem phim. \\
  \hline

  2 & \textbf{Dương Trí Phúc} \newline
  MSSV: N22DCCN061 \newline
  (Thành viên) &
  \textbf{1. Nhiệm vụ chính:} \newline
  \textbullet\ \textbf{Trang chủ (Home):} Thiết kế cấu trúc giao diện trang chủ (\texttt{HomeFragment}) phân vùng banners, phim ảnh và thể thao; lập trình REST API lấy dữ liệu tổng hợp trang chủ (\texttt{/api/v1/films/home}). \newline
  \textbullet\ \textbf{Shorts (Video ngắn):} Xây dựng giao diện video ngắn dạng cuộn dọc (\texttt{ShortsFragment}) sử dụng \texttt{ViewPager2}, tích hợp thả tim và bình luận; viết REST API Shorts hỗ trợ danh sách, thích và lưu bình luận dạng JSON/JSONB. \newline
  \textbf{2. Nhiệm vụ phối hợp:} \newline
  \textbullet\ \textbf{Phát video \& ExoPlayer:} Tích hợp trình phát ExoPlayer phát video streaming, xử lý hàm phân giải loopback IP \texttt{resolveLocalhostUrl()} cho thiết bị Emulator. \newline
  \textbullet\ \textbf{Tải ngoại tuyến (Offline):} Lập trình giao diện quản lý tải offline, lớp quản lý lưu trữ dữ liệu tải ngoại tuyến (\texttt{OfflineDownloadManager}) bằng SharedPreferences và Gson. \\
  \hline

  3 & \textbf{Nguyễn Anh Kha} \newline
  MSSV: N22DCCN038 \newline
  (Thành viên) &
  \textbf{1. Nhiệm vụ chính:} \newline
  \textbullet\ \textbf{Phim lẻ \& Phim bộ:} Thiết kế giao diện chi tiết phim (\texttt{FilmDetailActivity}) hiển thị thông tin phim lẻ và danh sách tập phim bộ; viết REST API truy xuất thông tin chi tiết phim và tập phim liên kết (\texttt{/api/v1/films/\{id\}}). \newline
  \textbullet\ \textbf{Quản trị Admin:} Xây dựng Dashboard điều hướng Admin (\texttt{AdminDashboardActivity}), danh sách quản lý phim (\texttt{AdminFilmListActivity}), biểu mẫu dialog thêm/sửa phim bộ và tập phim (\texttt{FilmFormDialogFragment}); lập trình REST API CRUD phim, tập phim, danh mục và cấu hình phân quyền Security (\texttt{SecurityConfig}). \newline
  \textbf{2. Nhiệm vụ phối hợp:} \newline
  \textbullet\ \textbf{Xác thực tài khoản (Auth):} Xây dựng giao diện Đăng ký (\texttt{RegisterActivity}), Đăng nhập (\texttt{LoginActivity}), Quên mật khẩu (\texttt{ForgotPasswordActivity}); lập trình REST API xác thực JWT và cơ chế Spring Mail gửi mã đặt lại mật khẩu. \newline
  \textbullet\ \textbf{Đa ngôn ngữ (Localization):} Triển khai cấu trúc đa ngôn ngữ Anh - Việt. \\
  \hline

\end{longtable}
